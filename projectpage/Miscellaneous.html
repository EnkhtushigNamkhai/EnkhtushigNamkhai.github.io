Christmas Robot

This project, which is unified around the theme of Christmas, features four key parts that allow the user to interact with its components. By touching a thermistor, users are able to activate a set of red LEDs and invoke a volume-controlled sound system that plays “We Wish You a Merry Christmas.” In the case that users are too lazy to walk over to the display, our device also provides an interactive mobile feature that allows one to turn on the display by sending a signal from a mobile device to the msp430 microcontroller via Bluetooth. It is our hope that this project will enable users to further enhance the joy of Christmas within their own homes. For more detailed documentation please go here.
